ADMIN SQL LOCAL PLUGIN
============================

Admin SQL is a Moodle 2.X local plugin that allows admins to execute SQL statements.

Web: https://github.com/IOC/moodle-local_adminsql


Install
=======

Download code from gitHub, and copy inside directory MOODLE_PATH/local/adminsql
